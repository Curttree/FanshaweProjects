### Build Instructions

## Running the Project
- Controls:
    - Arrow Keys - Movement
    - Left Mouse Button - Fire
    - Space Bar - Jump

- Colours for each steering behaviour are as follows:

    - Enemy Type A :
        - Seek : Black (should not ever see though due to first person perspective )
        - Flee : Red
    - Enemy Type B:
        - Pursue : Deep Blue
        - Evade : Sky Blue
    - Enemy Type C:
        - Approach : Green
        - (When within minimum approach radius) : Cyan
    - Enemy Type D:
        - Wander : Silver
        - Idle : White
